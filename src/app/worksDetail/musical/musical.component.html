About the work

This project is one of the many wonderful works which hacks IKEA products and provide alternative ideas to people. ‘Musical Train’ started off from IKEA’s LILLABO set, which consist of wooden train tracks and trains for child to play with. Although it is a simple child’s toy, the fine quaility, with smooth movement of the train over the rails, surprised us.

With its beautiful aesthetic and graceful movement, this product can be used as a silence-keeper in a public place, as well as a moodlight. So we got an idea to use this LILLABO set to make it into more interactive toy, which children could learn musical code and also make a melody out of it.

We first made color coded music chips which fit nicely with LILLABO train tracks. And then made small train with sensors underneath, which could catch the color of the tracks. With this simple extension, LILLABO set became not just an wooden toy but also an interactive learning tool which childrens could try various options to play with and learn during the process.

Why I chose this work

This work was, although simple in idea, quite challenging in reality because every sensor, power and circuit was to be inside a small IKEA-style-toy-train.

Underneath is the simplified structural drawing of the product. It shows briefly how music is created by this train. First, the color sensor beneath the train sense the color change of the rail. This is then signaled to the arduino processor. And through mp3 module, the appropriate music note is played.
